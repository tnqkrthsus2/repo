# 지각률 분석
## 개발환경
### Rstudio
## 참고
### 경제학원론 출결표(part.csv)
## 목적
### 각 주차별 출결의 변화 파악
## 데이터 전처리 과정
### 인터넷 접속 오류로, 한번의 수업에서 여러번의 접속 기록이 있는 학생의 접속 시간을 취합
### 수업 결석에 대한 기준을 교수님께서 따로 공표받지 못했으모로, 매 수업별 접속시간의 90%에 미달하는 접속시간을 가진 학생을 지각으로 처리
